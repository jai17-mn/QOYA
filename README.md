# QOYA
QOYA est un assistant dédié aux propriétaires de chats pour simplifier leur quotidien. Le site propose des outils de suivi comme la transition de croquettes, le calcul des portions, la gestion des stocks, les rappels d'entretien et de santé, ainsi qu'une bibliothèque de conseils pour mieux comprendre et prendre soin de son compagnon.
